# Stellarnet Frontend 🌌

Interface moderna e responsiva do sistema de gestão de estoque Stellarnet.

## 🚀 Tecnologias

- **Next.js 15+**: Framework React para produção.
- **TailwindCSS 4**: Estilização baseada em utilitários com suporte a Neumorfismo.
- **Framer Motion**: Biblioteca para animações avançadas de UI.
- **Context API**: Gerenciamento de estado global (Auth, Chat, Tema).
- **Internationalization (i18next)**: Suporte multi-idioma.

## 🎨 Design System

O frontend segue uma estética premium com:
- **Vidomorfismo (Glassmorphism)** em painéis e modais.
- **Neumorfismo** em botões e cards de controle.
- **Modo Escuro Nativo** com paletas de cores azuladas profundas e cinza antracite.

## 🛠️ Comandos Disponíveis

No diretório `/frontend`:

```bash
# Iniciar servidor de desenvolvimento
npm run dev

# Construir para produção
npm run build

# Iniciar servidor de produção
npm run start
```

---
Acesse a aplicação principal em: `http://localhost:3000`
